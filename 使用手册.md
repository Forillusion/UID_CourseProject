## 二、核心工作流：道路提取（OR1）

道路是整个系统的基础，必须先完成道路提取才能加载车辆。

### 完整流程

```
① 点击「提取骨架」
   → 进入「点选输入」模式（模式标签变红）
   → 右侧地图上开始画道路

② 左键点击地图画道路
   → 每次左键添加一个节点，自动与前一个节点连线（红色粗线）
   → 实时显示已画的道路骨架

③ 右键点击
   → 结束当前折线（可以开始画新的折线）
   → 注意：右键不退出提取模式，只是结束当前折线

④ 如需擦除某条线段
   → 点击「擦除道路」按钮（模式标签变紫，光标变圆圈）
   → 点击要删除的线段（整条删除）
   → 再次点击「擦除道路」（或按钮显示「返回点选」时点击）回到点选模式
   → 擦除线段时，该线段上的车辆也会被自动删除

⑤ 点击「提取结束」
   → 系统自动生成道路掩膜
   → 红色骨架消失，蓝色道路区域显示
   → 「擦除道路」按钮变为不可用
   → 模式标签变为「提取完成」（蓝色）

⑥ 如需继续编辑道路
   → 再次点击「提取骨架」
   → 蓝色道路隐藏，红色骨架重新显示
   → 可以继续画线或擦除
   → 完成后再次点击「提取结束」
```

### 道路宽度调整

- 拖动「道路半宽」滑块（1-15 像素）调整道路宽度
- 在「提取完成」状态下调整会实时更新蓝色道路显示
- 推荐值：2-5 像素（对应现实约 3.4-8.5 米）

### 清空道路

- 点击「清空道路」删除全部骨架、掩膜和**所有车辆**
- 状态回到「空闲」

## 三、智能车 (IV) 操作

### 前提
必须先完成道路提取（状态为「提取完成」），否则无法加载车辆。

### 加载车辆
1. 点击「加载车辆」
2. 在地图上**道路区域**（蓝色部分）点击放置车辆
3. 如果点击不在道路上，会弹出错误提示
4. 车辆以绿色矩形显示，前端黄色段为车头（与车身同宽）
5. 放置后自动回到空闲模式

### 调整朝向
1. 在车辆下拉框中选择要调整的车辆
2. 拖动「朝向」滑块（0-360°）
3. 地图实时更新车辆方向

### 移除车辆
1. 在车辆下拉框中选择要移除的车辆
2. 点击「移除车辆」

### 报告位置
- 点击「报告位置」弹出对话框，显示所有车辆的现实世界坐标（米）和朝向

## 四、测量功能

### 两点测距
1. 点击「两点测距」
2. 在地图上点击第一个点
3. 点击第二个点
4. 青色连线绘制，距离显示在面板上（单位：米）

### 轨迹测量
1. 点击「轨迹测量」
2. 连续点击多个点（青色连线实时绘制）
3. 右键结束
4. 总长度显示在面板上

### 清除测量标记
- 点击「清除测量标记」移除所有测量连线和节点

---

## 五、地图旋转

- 拖动「(有bug)旋转角度」滑块（-180° 到 180°）
- 地图连同骨架/道路/车辆一起旋转显示
- ⚠️ **已知 bug**：旋转状态下，鼠标点击的坐标转换可能不准确。建议在旋转前完成所有需要精确点击的操作（画骨架、加载车辆、测量等），旋转仅用于观察视角。

---

## 六、坐标显示

- 鼠标在地图上任意位置按下时，左下角显示该点的现实世界坐标
- 坐标原点为地图左下角，X 向右，Y 向上
- 1 像素 = 1.7 米

---

## 七、状态说明

| 模式 | 标签颜色 | 说明 | 可用操作 |
|------|----------|------|----------|
| 空闲 | 绿色 | 初始状态 | 提取骨架 |
| 点选输入 | 红色 | 正在画骨架 | 左键画点、右键结束折线、提取结束、擦除道路 |
| 擦除 | 紫色 | 擦除线段 | 点击线段删除、返回点选、提取结束 |
| 提取完成 | 蓝色 | 道路已就绪 | 提取骨架(继续编辑)、加载车辆、测量 |

---

## 八、注意事项

1. **操作顺序**：必须按「提取骨架 → 提取结束 → 加载车辆」的顺序操作。
2. **道路校验**：车辆只能加载在道路上，非道路区域会被拒绝。
3. **擦除影响**：擦除道路线段会同时删除该线段上的车辆。
4. **清空影响**：清空道路会删除所有骨架、掩膜和车辆。
5. **纯手写实现**：所有图像操作（旋转、叠加、画车）均为手动矩阵操作，未使用高级内置函数。
