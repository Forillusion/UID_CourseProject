# UI Structure

Run the project with:

```matlab
main
```

The project is based on the GitHub UI framework.

In the main window, optional work is grouped behind entry buttons:

- `OR1 道路骨架` opens the OR1 road skeleton menu in a separate window.
- `OR2 预留` to `OR5 预留` are reserved for later optional work.

The OR1 detailed controls such as `提取骨架`, `提取结束`, `擦除道路`, and `清空道路`
are not shown directly in the main panel. They appear only after clicking
`OR1 道路骨架`.
