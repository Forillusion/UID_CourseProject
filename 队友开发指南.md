## 二、全局状态 S 结构（最重要！）

所有数据通过 `setappdata(fig,'S',S)` / `S = getappdata(fig,'S')` 传递。
**你的代码中读写数据必须遵循此模式。**

```matlab
function yourFunction(fig, ...)
    S = getS(fig);          % 读取全局状态
    % ... 使用 S.xxx ...
    S.yourField = xxx;       % 修改
    setS(fig, S);            % 写回（必须有这一步！否则其他函数看不到改动）
end
```

## 三、坐标系约定（易错点）

```
图像矩阵: img(row, col, channel)
  row = 1 在顶部（Y 小），row = mapH 在底部（Y 大）
  col = 1 在左侧（X 小），col = mapW 在右侧（X 大）

世界坐标: 原点在图像左下角，Y 向上
  worldX = col × 1.7
  worldY = (mapH - row) × 1.7

转换函数（已内置）：
  [wx, wy] = px2world(fig, col, row)   % 像素→世界
  [col, row] = world2px(fig, wx, wy)   % 世界→像素
```

## 五、如何在地图上画自定义内容

**核心模式**：用 `buildBaseMap` 获取基础图 → 叠加你的内容 → 赋给 `S.mapDisplay` → `refreshView`

```matlab
function drawYourStuff(fig)
    S = getS(fig);
    if isempty(S.mapOrigin), return; end

    % 1. 获取基础图（原图 + 骨架/蓝路 + 车辆，按 sketchState 自动决定）
    map = buildBaseMap(fig);
    [H, W, ~] = size(map);

    % 2. 在这里叠加你的内容
    %    直接修改 map 的像素值
    %    例: map(r, c, :) = uint8([255 0 0]);  % 红色

    % 3. 应用旋转 + 显示
    if S.rotDeg ~= 0
        S = getS(fig);
        S.mapDisplay = rotateMap(map, S.rotDeg);
        S.rotSize = [size(S.mapDisplay,1), size(S.mapDisplay,2)];
    else
        S.mapDisplay = map;
        S.rotSize = [];
    end
    setS(fig, S);
    refreshView(fig);
end
```

## 七、骨架工作流状态机

骨架提取有 4 个状态，由 `setSketchState` 统一管理：

```
idle ──「提取骨架」──→ sketching ──「提取结束」──→ finalized
                         ↑──「提取骨架」──│
                         │                  ↑
                         └──「擦除道路」──→ erasing ──「返回点选」──┘
```

| 状态 | 显示内容 | 说明 |
|------|----------|------|
| `idle` | 仅原图+车辆 | 初始/清空状态 |
| `sketching` | 原图+红色骨架+车辆 | 正在画线，左键画点右键结束折线 |
| `erasing` | 原图+红色骨架+车辆 | 擦除模式，点击线段删除（圆形光标） |
| `finalized` | 原图+蓝色道路区+车辆 | 道路已就绪，可加载车辆 |

**重要**：你的 OR 任务如果需要道路数据，请检查 `S.sketchState == 'finalized'` 且 `~isempty(S.roadMask)`。

```matlab
function onBtnShowLocal(fig)
    S = getS(fig);
    if isempty(S.vehicles)
        uialert(fig, '请先加载车辆。', '提示');
        return;
    end
    v = S.vehicles(1);   % 以第一辆车为中心
    radius_m = S.handles.radiusEdit.Value;
    radius_px = radius_m / S.scale;

    map = buildBaseMap(fig);
    [H, W, ~] = size(map);

    % 创建圆形 mask
    [rr, cc] = meshgrid(1:W, 1:H);
    circleMask = (cc - v.cx).^2 + (rr - v.cy).^2 <= radius_px^2;

    % 圆外区域置灰
    for ch = 1:3
        tmp = map(:,:,ch);
        tmp(~circleMask) = 100;   % 灰色
        map(:,:,ch) = tmp;
    end

    S.mapDisplay = map;
    S.rotSize = [];
    setS(fig, S);
    refreshView(fig);
    setStatus(fig, sprintf('局部地图：半径 %.0f 米 (%.0f 像素)', radius_m, radius_px));
end
```

缩放因子：修改 `S.vehicles(i).dispScale` 后调用 `refreshDisplay(fig)`。

### OR3：车辆加载时自动对齐方向

**任务要求：**
1. 车辆加载到道路点后，朝向自动与道路走向对齐
2. 支持「车头始终朝上」模式（地图围绕车辆反向旋转）

**实现方案——自动对齐：**

在 `handleLoadIVClick` 中，加载车辆时计算道路方向：
```matlab
function angle = getRoadDirection(col, row, S)
    % 在骨架边上找最近的边，用其方向
    P = [col, row];
    bestAngle = 0; bestDist = inf;
    for i = 1:size(S.sk.edges, 1)
        ni = S.sk.edges(i, 1); nj = S.sk.edges(i, 2);
        A = S.sk.nodes(ni, :); B = S.sk.nodes(nj, :);
        d = ptToSegDist(P, A, B);
        if d < bestDist
            bestDist = d;
            bestAngle = atan2d(B(2) - A(2), B(1) - A(1));
        end
    end
    angle = mod(bestAngle, 360);
end
```

**车头朝上模式：**

面板加一个开关按钮，开启后用 `rotateMap(S.mapDisplay, -vehicleAngle)` 旋转地图，车辆画在画布中心。

### OR4：虚拟街景生成

**任务要求：**
- 给定任意道路点及朝向，生成虚拟街景图像
- 基于平面地图和三维映射原理

**实现方案（透视投影）：**

```matlab
function scene = genStreetView(map, roadCol, roadRow, heading, scale)
% 生成虚拟街景（针孔相机模型）
%  map: 地图矩阵
%  roadCol, roadRow: 道路点像素坐标
%  heading: 朝向（度）
%  scale: 1.7 (米/像素)

    camHeight = 2.0;    % 相机高度（米）
    fov = 60;           % 视野角度
    imgW = 400; imgH = 300;
    scene = uint8(zeros(imgH, imgW, 3));

    th = deg2rad(heading);
    f = (imgW / 2) / tand(fov / 2);   % 焦距

    for v = 1:imgH       % 每一行对应一个深度
        for u = 1:imgW   % 每一列对应一个水平角度
            % 射线方向
            dx = (u - imgW/2) / f;
            dy = 1;
            dist_m = camHeight * f / (v - imgH/2 + eps);
            if dist_m < 0 || dist_m > 200
                continue;
            end
            dist_px = dist_m / scale;
            % 世界坐标偏移
            worldX = dist_px * dx;
            worldY = dist_px * dy;
            % 旋转到朝向
            rotX =  worldX * cos(th) + worldY * sin(th);
            rotY = -worldX * sin(th) + worldY * cos(th);
            % 映射回地图像素
            mapCol = round(roadCol + rotX);
            mapRow = round(roadRow - rotY);  % Y 向上 = row 减小
            if mapCol >= 1 && mapCol <= size(map,2) && ...
               mapRow >= 1 && mapRow <= size(map,1)
                scene(v, u, :) = map(mapRow, mapCol, :);
            end
        end
    end
end
```

注意：教授说会在练习课补充相关知识，建议等课上完再优化。

### OR5：路径规划

**任务要求：**
1. 给定地图任意两点，找到离它们最近的道路点
2. 计算两道路点之间的最短道路路径
3. 手写 Dijkstra 算法（不能调包）

**实现方案：**

                     'ButtonPushedFcn', @(s,e) onBtnPlanPath(fig));

                      'ButtonPushedFcn', @(s,e) onBtnClearPath(fig));

**步骤 1：找最近道路点**
```matlab
function [bestC, bestR] = findNearestRoad(col, row, S)
    [rr, cc] = find(S.roadMask);
    dists = (cc - col).^2 + (rr - row).^2;
    [~, idx] = min(dists);
    bestC = cc(idx); bestR = rr(idx);
end
```

**步骤 2：手写 Dijkstra**
```matlab
function [path, totalDist] = dijkstra(nodes, edges, startIdx, endIdx)
    n = size(nodes, 1);
    % 构建邻接表
    adj = cell(n, 1);
    for i = 1:size(edges, 1)
        u = edges(i, 1); v = edges(i, 2);
        w = norm(nodes(u,:) - nodes(v,:));
        adj{u} = [adj{u}; v w];
        adj{v} = [adj{v}; u w];
    end
    % Dijkstra
    dist = inf(n, 1);
    prev = zeros(n, 1);
    visited = false(n, 1);
    dist(startIdx) = 0;
    for iter = 1:n
        tempDist = dist;
        tempDist(visited) = inf;
        [d, u] = min(tempDist);
        if isinf(d), break; end
        visited(u) = true;
        if u == endIdx, break; end
        for j = 1:size(adj{u}, 1)
            v = adj{u}(j, 1);
            w = adj{u}(j, 2);
            if ~visited(v) && dist(u) + w < dist(v)
                dist(v) = dist(u) + w;
                prev(v) = u;
            end
        end
    end
    totalDist = dist(endIdx);
    % 回溯路径
    if isinf(totalDist)
        path = [];
        return;
    end
    path = endIdx;
    while prev(path(1)) ~= 0
        path = [prev(path(1)), path];  %#ok<AGROW>
    end
end
```

**步骤 3：可视化路径**

用 `bresenham` 画路径连线（绿色粗线），叠加到 `buildBaseMap` 的结果上。

## 十、常见陷阱

1. **uint8 溢出**：`uint8(200) + uint8(100) = 255`（不是 300）。运算前先 `double()`，做完再 `uint8()`。
2. **Y 轴方向**：图像 row 向下，世界 Y 向上。显示用 `YDir='reverse'`。
3. **改了 S 必须写回**：`S = getS(fig); S.xxx = yyy; setS(fig, S);` 三步缺一不可。
4. **mapOrigin 只读**：叠加内容时从 `buildBaseMap(fig)` 获取基础图，永远不要直接改 `S.mapOrigin`。
5. **size 对 3D 数组**：用 `[H,W,~] = size(img)`，不要用 `[H,W] = size(img)`（后者 W = 维度2×维度3 乘积）。
6. **local function**：所有函数都写在 main.m 内部，外部脚本无法直接调用。测试时在测试脚本里放函数副本。
7. **控件属性**：uifigure 控件用 `FontColor` 不是 `ForegroundColor`，用 `.Text` 不是 `.String`（dot 写法）。
8. **编码**：修改 main.m 时务必保持 UTF-8 编码，不要用 PowerShell 的 Set-Content（会损坏中文）。
