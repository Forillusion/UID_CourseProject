## 二、全局状态 S 结构（最重要！）

所有数据通过 `setappdata(fig,'S',S)` / `S = getappdata(fig,'S')` 传递。
**你的代码中读写数据必须遵循此模式。**

```matlab
function yourFunction(fig, ...)
    S = getS(fig);          % 读取全局状态（等价于 getappdata(fig,'S')）
    % ... 使用 S.xxx ...
    S.yourField = xxx;       % 修改
    setS(fig, S);            % 写回（必须有这一步！否则其他函数看不到改动）
end
```

## 三、坐标系约定（⚠️ 易错点）

```
图像矩阵: img(row, col, channel)
  row = 1 在顶部（Y 小），row = mapH 在底部（Y 大）
  col = 1 在左侧（X 小），col = mapW 在右侧（X 大）

世界坐标: 原点在图像左下角，Y 向上
  worldX = col × 1.7
  worldY = (mapH - row) × 1.7

转换函数（已内置）：
  [wx, wy] = px2world(fig, col, row)   % 像素→世界
  [col, row] = world2px(fig, wx, wy)   % 世界→像素
```

## 四、如何添加你的按钮

在 `buildPanel` 函数中找到你的 OR 分组，添加按钮：

```matlab

r = r + 1;

                        'ButtonPushedFcn', @(s,e) onBtnYourFeature(fig));
```

然后在文件中添加回调函数：
```matlab
function onBtnYourFeature(fig)
    S = getS(fig);
    S.mode = 'yourMode';    % 如果需要鼠标交互
    setS(fig, S);
    setStatus(fig, '你的提示文字。');
end
```

如果需要鼠标点击交互，在 `onMouseDown` 的 `switch S.mode` 中添加：
```matlab
case 'yourMode'
    handleYourClick(fig, col, row);
```

## 五、如何在地图上画自定义内容

**核心模式**：从 `S.mapOrigin` 复制 → 叠加你的内容 → 赋给 `S.mapDisplay` → 调用 `refreshView`

```matlab
function drawYourStuff(fig)
    S = getS(fig);
    S.mapDisplay = S.mapOrigin;   % 1. 从原图复制
    setS(fig, S);

    % 2. 叠加骨架（保持一致性）
    if ~isempty(S.sk.edges)
        drawSkeletonOnMap(fig);
    end

    % 3. 叠加车辆
    S = getS(fig);
    for i = 1:numel(S.vehicles)
        S.mapDisplay = drawIV(S.mapDisplay, ...
            S.vehicles(i).cx, S.vehicles(i).cy, ...
            S.vehicles(i).angle, S.vehicles(i).dispScale, ...
            S.mapW, S.mapH, S.scale);
    end

    % 4. ★ 在这里叠加你的内容 ★
    %    直接修改 S.mapDisplay 的像素值
    %    例: S.mapDisplay(r, c, :) = uint8([255 0 0]); % 红色

    % 5. 显示
    setS(fig, S);
    refreshView(fig);
end
```

### OR2：缩放与局部可视化 ★★☆

**任务要求：**
1. 可调缩放因子放大 IV 的可视化尺寸
2. 以 IV 为中心，显示指定半径的圆形局部地图
3. 半径必须严格吻合指定范围

**实现方案：**

```
面板新增：
  - "IV缩放倍数" 滑块 (1-10)
  - "局部半径(米)" 输入框
  - "显示局部地图" 按钮
  - "显示全部地图" 按钮（恢复）

逻辑：
  1. 选择一辆车（用 S.handles.ivDropdown）
  2. 半径(米) → 半径(像素) = 半径 / 1.7
  3. 以车辆 (cx, cy) 为圆心，radius 为半径
  4. 创建圆形 mask：对每个像素算到圆心距离 < radius
  5. 在 mapDisplay 上，把圆外的区域设为黑色（或灰色）
  6. 同时用新的 dispScale 画车

关键代码片段：
  radius_px = radius_m / S.scale;
  [rr, cc] = meshgrid(1:S.mapW, 1:S.mapH);
  circleMask = (cc - cx).^2 + (rr - cy).^2 <= radius_px^2;
  % 圆外置灰
  for ch = 1:3
    tmp = S.mapDisplay(:,:,ch);
    tmp(~circleMask) = 128;   % 灰色
    S.mapDisplay(:,:,ch) = tmp;
  end
```

### OR3：车辆加载时自动对齐方向 ★★☆

**任务要求：**
1. 车辆加载到道路点后，朝向自动与道路走向对齐
2. 支持"车头始终朝上"模式（地图围绕车辆反向旋转）

**实现方案：**

```
1. 自动对齐：
   - 车辆加载点在 roadMask 上
   - 找到该点附近的道路方向：
     方案A: 在骨架上找最近的 edge，计算 edge 的角度
     方案B: 在 roadMask 上取一个小窗口，用 PCA 求主方向
   - 设置车辆 angle = 道路方向

   代码（方案A）:
   function angle = getRoadDirection(col, row, S)
       % 在骨架边上找最近的边
       P = [col, row];
       bestAngle = 0; bestDist = inf;
       for i = 1:size(S.sk.edges,1)
           ni = S.sk.edges(i,1); nj = S.sk.edges(i,2);
           A = S.sk.nodes(ni,:); B = S.sk.nodes(nj,:);
           d = ptToSegDist(P, A, B);
           if d < bestDist
               bestDist = d;
               bestAngle = atan2d(B(2)-A(2), B(1)-A(1));
           end
       end
       angle = mod(bestAngle, 360);
   end

2. 车头朝上模式：
   - 用 rotateMap(S.mapDisplay, -vehicleAngle) 旋转地图
   - 车辆画在画布中心
   - 面板加一个开关 "车头朝上模式"
```

### OR4：虚拟街景生成 ★★★★

**任务要求：**
- 给定任意道路点及朝向，生成虚拟街景图像
- 基于平面地图和三维映射原理

**实现方案（简化版）：**

```
核心原理：透视投影
1. 定义虚拟相机参数：
   - 相机高度 h（如 2 米）
   - 视野角度 fov（如 60°）
   - 远裁剪面 far（如 100 米）

2. 对街景图的每个像素，发射一条射线：
   - 计算射线在世界中的方向
   - 射线与地面(y=0)的交点
   - 从交点查地图原图的颜色
   - 赋给街景图

3. 关键公式（针孔相机模型）：
   对于街景图像素，对应的世界坐标：
   worldX = camX + (u - cx) / f * dist
   worldZ = camZ + dist
   其中 dist = h * f / (v - cy)

   (h=相机高度, f=焦距, cx,cy=光心)

注意：教授说会在练习课补充，建议等课上完再做。
可以先用简单的前方拉伸效果做 demo。
```

### OR5：路径规划 ★★★

**任务要求：**
1. 给定地图任意两点，找到离它们最近的道路点
2. 计算两道路点之间的最短道路路径
3. 手写 Dijkstra 算法（不能调包）

**实现方案：**

```
面板新增：
  - "设置起点" 按钮
  - "设置终点" 按钮
  - "规划路径" 按钮
  - "清除路径" 按钮

步骤1：找最近道路点
  function [bestC, bestR] = findNearestRoad(col, row, S)
      % 在 roadMask 上找离 (col,row) 最近的 true 像素
      [rr, cc] = find(S.roadMask);
      dists = (cc - col).^2 + (rr - row).^2;
      [~, idx] = min(dists);
      bestC = cc(idx); bestR = rr(idx);
  end

步骤2：构建图（用骨架节点+边）
  - 图的节点 = S.sk.nodes
  - 图的边 = S.sk.edges，权重 = 节点间欧氏距离
  - 额外把起点和终点作为临时节点连接到最近的骨架节点

步骤3：手写 Dijkstra
  function [path, dist] = dijkstra(nodes, edges, startIdx, endIdx)
      nNodes = size(nodes, 1);
      % 邻接表
      adj = cell(nNodes, 1);
      for i = 1:size(edges,1)
          u = edges(i,1); v = edges(i,2);
          w = norm(nodes(u,:) - nodes(v,:));
          adj{u} = [adj{u}; v w];
          adj{v} = [adj{v}; u w];
      end
      % Dijkstra
      dist = inf(nNodes, 1);
      prev = zeros(nNodes, 1);
      visited = false(nNodes, 1);
      dist(startIdx) = 0;
      for iter = 1:nNodes
          % 找未访问中距离最小的
          tempDist = dist;
          tempDist(visited) = inf;
          [d, u] = min(tempDist);
          if isinf(d), break; end
          visited(u) = true;
          if u == endIdx, break; end
          % 松弛邻居
          for j = 1:size(adj{u},1)
              v = adj{u}(j,1); w = adj{u}(j,2);
              if ~visited(v) && dist(u) + w < dist(v)
                  dist(v) = dist(u) + w;
                  prev(v) = u;
              end
          end
      end
      % 回溯路径
      path = endIdx;
      while prev(path(1)) ~= 0
          path = [prev(path(1)), path]; %#ok<AGROW>
      end
  end

步骤4：可视化路径
  - 用 bresenham 画路径连线（绿色粗线）
  - 在 drawYourStuff 中叠加
```

## 九、常见陷阱

1. **uint8 溢出**：`uint8(200) + uint8(100) = 255`（不是 300）。做运算前先 `double()`，做完再 `uint8()`。
2. **Y 轴方向**：图像 row 向下，世界 Y 向上。显示用 `YDir='reverse'`。
3. **改了 S 必须写回**：`S = getS(fig); S.xxx = yyy; setS(fig, S);` 三步缺一不可。
4. **mapOrigin 只读**：叠加内容时从 `S.mapOrigin` 复制，永远不要直接改它。
5. **size 对 3D 数组**：用 `[H,W,~] = size(img)`，不要用 `[H,W] = size(img)`（后者 W = 维度2×维度3 的乘积）。
6. **local function**：所有函数都写在 main.m 内部，外部脚本无法直接调用。测试时在测试脚本里放函数副本。
